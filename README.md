# skills

Public collection of agent skills written by Will Hampson.

## Structure

- `productivity/` — thinking, planning, writing, research, and personal operating-system skills.
- `engineering/` — software, systems, debugging, and automation skills.
- `personal/` — personal workflows, habits, family/home, and life-admin skills.

## Current skills

- `productivity/first-principles/` — interactive first-principles thinking grill for decomposing problems from fundamentals.

## Skill format

Each skill lives in its own directory and uses a `SKILL.md` file with YAML frontmatter followed by Markdown instructions.

Example:

```text
productivity/
  first-principles/
    SKILL.md
engineering/
  <skill-name>/
    SKILL.md
personal/
  <skill-name>/
    SKILL.md
```
